# Projeto FHS - Cálculo de Fatorial

## Descrição

Este projeto foi desenvolvido para a disciplina de Fundamentos de Hardware e Software (FHS).

O objetivo é calcular o fatorial de um número inteiro, representando a quantidade de maneiras possíveis de organizar kits de higiene contendo N produtos distintos.

Além do desenvolvimento da solução em Linguagem C e Python, o projeto foi utilizado na disciplina de Introdução à Engenharia de Computação (IEC) para aplicação dos conceitos básicos de SCRUM, Trello e GitHub.

## Tecnologias

- Linguagem C
- Python 3
- Git
- GitHub
- Trello
